-

- Клавиатура: [Microsoft Sculpt Ergonomic Keyboard](https://www.microsoft.com/accessories/en-us/products/keyboards/sculpt-ergonomic-desktop/l5v-00001)

- API-клиент: [Insomnia](https://insomnia.rest/)
